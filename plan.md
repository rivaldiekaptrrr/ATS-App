 Project Master Plan: SmartRecruit ATSTagline: Automated Hiring Pipeline with Real-time Candidate Engagement.1. Arsitektur Teknis (The Power Stack)Pemilihan stack ini didasarkan pada kecepatan pengembangan, performa tinggi, dan kemudahan untuk "dijual lepas".Frontend & Framework: Next.js 15 (App Router) dengan TypeScript.Styling: Tailwind CSS + Shadcn/UI (Untuk tampilan dashboard profesional).State Management: TanStack Query (React Query) untuk sinkronisasi data server yang cepat.Backend & Database (BaaS): Supabase.PostgreSQL untuk database.Supabase Auth untuk login HR.Supabase Storage untuk menyimpan file CV (PDF/Docx).Edge Functions untuk otomasi pengiriman email.Email Engine: Resend (API pengiriman email modern).Form Handling: React Hook Form + Zod (Validasi data yang sangat ketat).2. Modul Fitur UtamaA. Landing Page & Job Board (Public)Daftar lowongan kerja aktif dengan fitur pencarian dan kategori.Smart Application Portal:Opsi 1: Fast-Track (Upload CV): Drag-and-drop file, sistem mengekstrak metadata file.Opsi 2: Manual Form: Multistep form (Data diri -> Pengalaman -> Pendidikan -> Skill).Tracking Page: Halaman simpel di mana pelamar bisa memasukkan email + ID lamaran untuk melihat status terakhir mereka (tanpa perlu login).B. HR Command Center (Private/Admin)Dashboard Analytics: Statistik jumlah pelamar, persentase kelulusan di setiap tahap, dan waktu rata-rata proses rekrutmen.Kanban Pipeline: Board visual (seperti Trello) untuk menggeser kandidat dari satu tahap ke tahap berikutnya.Candidate Detail View: Menampilkan seluruh profil, pengalaman, dan preview file CV langsung di browser (tanpa download).Action History: Log aktivitas (misal: "HR A mengubah status kandidat B menjadi Lolos Interview pada jam 10:00").C. The Automation Engine (Notification System)Inilah nilai jual utama Anda. Setiap perpindahan status memicu hook pengiriman email.StatusTriggerNotifikasi (Email)AppliedData masuk"Konfirmasi: Lamaran Berhasil Diterima"ScreeningKlik Reject"Maaf, Anda belum lolos tahap Administrasi"InterviewKlik Pass"Selamat! Anda melaju ke tahap Interview. Cek jadwal berikut."InterviewKlik Reject"Maaf, Anda belum dapat lanjut dari tahap Interview"OfferingKlik Pass"Kabar Gembira! Kami menawarkan posisi ini kepada Anda (Lampiran PDF)"3. Struktur Database (Schema Design)TableFungsi UtamacompaniesData perusahaan (jika ingin dibuat sistem Multi-Tenant untuk dijual ke banyak PT).jobsDetail lowongan (posisi, gaji, deskripsi, persyaratan).applicantsData profil pelamar (Nama, Email, HP, Bio, URL_CV).applicationsPenghubung antara applicants dan jobs (Status saat ini, catatan HR, skor).notif_templatesMenyimpan template body email untuk setiap tahap agar bisa diubah-ubah oleh HR.4. Rencana Pengembangan (Development Phases)Fase 1: Minimum Viable Product (MVP)Setup Next.js & Supabase.Landing page lowongan kerja sederhana.Formulir apply (Upload & Manual).Dashboard tabel pelamar sederhana (Belum Kanban).Kirim email manual lewat tombol "Send Feedback".Fase 2: Automation & PipelineImplementasi Kanban Board (Drag-and-Drop).Integrasi Resend API untuk otomasi email berdasarkan trigger status.Fitur Preview PDF di dashboard HR.Fase 3: Premium Features (Value Add)AI Screening: Integrasi OpenAI API untuk merangkum CV pelamar secara otomatis.Custom Templates: HR bisa mengedit isi email notifikasi langsung dari dashboard.Export Data: Fitur download laporan rekrutmen ke Excel/PDF.5. Strategi "Dijual Lepas" (Resale Strategy)Agar aplikasi ini memiliki nilai tinggi saat dijual:Whitelabel Ready: Buat fitur untuk mengganti logo, warna tema, dan domain hanya melalui dashboard admin (Tanpa koding ulang).Clean Code & Documentation: Pastikan Antigravity menulis kode yang rapi dan memberikan dokumentasi API.Scalable Pricing: Jika dijual sebagai SaaS, Anda bisa membatasi jumlah lowongan (misal: Paket Free max 3 lowongan, Paket Pro unlimited).6. Instruksi Final untuk Antigravity (Prompt Kompleks)"Saya ingin Anda membangun aplikasi SmartRecruit ATS menggunakan Next.js 15, Supabase, Tailwind, dan Resend.Fitur Utama:Dashboard HR dengan Kanban Board untuk mengelola alur pelamar (Screening, Interview, Test, Offering).Sistem otomasi email: Kirim email 'Gagal' atau 'Lolos' secara otomatis SETIAP KALI HR mengubah status kandidat di dashboard. Isi email harus menyebutkan spesifik nama tahapannya.Portal pelamar yang memiliki opsi 'Upload CV' atau 'Isi Form Manual'.Gunakan Shadcn/UI untuk desain dashboard yang modern dan bersih.Arsitektur harus mendukung Multi-tenancy (bisa dipakai banyak perusahaan dalam satu database) agar aplikasi ini siap saya jual sebagai produk SaaS."
